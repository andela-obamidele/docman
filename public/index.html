    Welcome to DocMan API

    Welcome to the DocMan API - A Document Management System API! You can use our API to access DocMan API endpoints, which can give you information on how to register, login, add a new document and manage your document in our database.

    Authentication

    To get access to the document, you have to be registered: Register and get a token to be insert in the header as Authorization

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files...just save them to the `includes` folder and add them to the top of your `index.md`'s frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- You supplied an invalid payload.
    401 Unauthorized -- You have not set your authorization header.
    403 Forbidden -- You tried to access an endpoint that you don't have permision to access.
    404 Not Found -- You tried to access something that is not available.
    409 Conflict - You try to duplicate something that is not meant to be duplicated(i.e email, document title, username)
    500 Server Error -- This is our fault. The server is either going through maintenance. Less often, this might be from your side
